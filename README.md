# vuejs-demo-v2

### 技术要点
- Vue.js 2.6.11
- Vue CLI 4.4.0
- Vuex 3.5.1
- VueRouter 3.3.4

### 学习要点
- bootstrap-sass 和 font-awesome 的使用；
- 组件的基本使用；
- 数据绑定的两种方式；
- 列表的渲染；
- class 和 style 的绑定；
- 事件的基本使用；
- 指令的基本使用；

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
